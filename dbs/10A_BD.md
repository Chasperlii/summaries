# Big Data

## Welche 3 Definitionen der V's bei Big Data gibt es?
* Gartner Group: 3V
    * Variety
    * Volume
    * Velocity
* IBM: 4V
    * _Gartner Group_ + Veracity
* Demchenko 2013: 5V
    * _IBM_ + Value
* Demchenko 2014: 6V
    * _Demchenko 2013_ + Variability

## Aus welchen vier Schritten besteht der Daten-Lebenszyklus?
* Collection:
    * This stage gathers and stores data in its original form (i.e., raw data.)
* Preparation:
    * This stage involves the collection of processes that convert raw data into cleansed, organized information
* Analysis:
    * This stage involves the techniques that produce synthesized knowledge from organized information
* Action:
    * This stage involves processes that use the synthesized knowledge to generate value for the enterprise
